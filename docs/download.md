# download module

Provides functions for downloading geospatial data from various sources, including NAIP aerial imagery, Sentinel-2 satellite imagery, Landsat data, Overture Maps features, and Microsoft Planetary Computer datasets. This module handles authentication, spatial queries, and format conversion to simplify data acquisition for analysis workflows.
