# pointcloud module

Provides semantic segmentation of 3D point clouds (LAS/LAZ files) using RandLA-Net via Open3D-ML. Supports inference with pre-trained models for outdoor driving, urban mapping, and indoor scanning scenarios, as well as fine-tuning on custom labeled point cloud data.
