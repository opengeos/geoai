# TESSERA module

Provides integration with TESSERA (Temporal and Spectral Satellite Embeddings for Remote Analysis) for generating satellite imagery embeddings. This module leverages pre-trained TESSERA models to produce dense feature representations from multi-spectral and multi-temporal satellite data, supporting tasks such as land cover mapping, crop classification, and environmental monitoring.
