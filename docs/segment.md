# segment module

Provides GroundedSAM and CLIP-based segmentation for geospatial imagery. This module combines text-prompt-driven object detection with the Segment Anything Model (SAM) to perform zero-shot semantic and instance segmentation, allowing users to segment objects in satellite and aerial images using natural language descriptions.
